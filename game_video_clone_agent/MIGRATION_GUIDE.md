# 🎬 生产线搬家（新电脑部署）全流程指南

老板，只要按照这四步走，您的生产线就能在新电脑上完成“原地满血复活”。

### 第一步：旧电脑“打包”
1. 拷贝整个 `game_video_clone_agent` 文件夹到 U 盘。

### 第二步：新电脑“安家”
1. 将文件夹从 U 盘拷贝到新电脑硬盘（**千万别在 U 盘里直接跑，速度会很慢**）。
2. 安装 Python 3.10+。
   - **关键提示**：安装时一定要勾选 `Add Python to PATH`。

### 第三步：一键配置环境
1. **双击运行 `一键部署工具.bat`**。
   - 它会自动帮您安装所有 AI 依赖包。
   - 如果它报错或闪退，请右键选择“以管理员身份运行”。

### 第四步：最后两件事（必做！）
1. **百度网盘授权**：双击 `authorize_baidu.bat`，扫码登录您的百度网盘并填入授权码。
2. **正式上线**：双击 `一键飞书机器人上线.bat`。
   - 看到提示成功后，窗口会停住。您关掉它即可，程序已经在后台隐身运行了。

---
### 🚨 常见问题 Q&A
*   **如何验证？**：在手机飞书上给机器人发个“状态”，它回您了就说明通了。
*   **如何关掉程序？**：如果您想彻底关掉它，只需在文件夹里随便开个黑窗口敲：`wmic process where "commandline like '%%feishu/%%'" call terminate`。
*   **视频合成报错？**：请确保新电脑安装了 FFmpeg。
