【当前阶段：中年/壮年】角色呈现经典的圆豆头、火柴人四肢造型。身披厚重的深色长款皮质风衣，头戴宽檐黑帽和标志性的白色鸟嘴面具，手持一根用于检查病患的长木手杖。透过面具的深色玻璃护目镜，能清晰看到一双疲惫、专注且透着坚毅的眼睛。

【阶梯式演变设计】
- [child]：幼年时期。身材矮小，头身比更大，脸庞稚嫩无面具，穿着简短朴素的粗麻布中世纪平民束腰外衣。手里拿着一个手工雕刻的鸟嘴面具小木雕玩具，眼神纯真但带着倔强。
- [elderly]：老年时期。体态明显佝偻，火柴人四肢显得更加迟缓无力。宽檐帽下露出稀疏的白发，鸟嘴面具布满划痕与缝补的金属铆钉，黑风衣变得褪色且边缘破损。原本用于指点病患的手杖变成了他沉重倚靠的拐杖，眼神沧桑且深邃。

【神髓一致性】
无论处于哪个年龄段，角色那双略带下垂、专注且倔强的眼睛始终如一（中年和老年时透过护目镜展现）。此外，“木制棍状物”（幼年的木雕玩具 -> 中年的木手杖 -> 老年的木拐杖）作为伴随一生的视觉锚点，保持了强烈的身份连贯性。
